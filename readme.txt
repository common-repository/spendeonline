Das Plugin ermöglicht den Zugriff auf SpendeOnline.org, die Internet-Plattform zum Sammeln von Spenden.

== Description ==

Das Plugin stellte ein WordPress-Plugin bereit, um auf SpendeOnline.org zuzugreifen.

SpendeOnline.org ist eine Internet-Plattform zum Sammeln von Spenden.
??ber ein Web-Formular kann der Spender seine Spende durchführen.

Im Online-Portal können Sie:

*   Spendenaktionen anlegen
*   Spendenaktionen auswerten
*   Spender verwalten, z.B. per Mail und/oder Serienbrief anschreiben
*   Zahlungsverwaltung: Bankeinzüge an die Bank geben, ??berweisungen verfolgen
*   Automatisch bei Zahlungseingang die Spendenquittung als PDF versenden.
*   und vieles mehr!

Details:

Spenden-Formular: Binden Sie das fertige Spenden-Formular auf Ihrer Webseite ein und sammeln Sie ganz einfach Spenden online.

*   Jeder Besucher kann direkt spenden.
*   Es kann aus konfigurierten Zahlungsweisen ausgewählt werden:
*   ??berweisung, Bankeinzug, Sofortüberweisung, Paypal.
*   Der Spender bekommt je nach Zahlungseingang die Spendenquittung (oder bei spezieller Einstellung auch sofort nach der Spende).
*   Der Spender kann, sofern angelegt, aus Spendenaktionen auswählen und damit zweckgebunden spenden.

Spender-Verwaltung: Verwalten Sie Ihre Spendenaktionen und Ihre Spender online. Werten Sie Ihre Aktionen aus.

*   Die Spender-Verwaltung und die weiteren Punkte sind interne Bereiche, auf die nur Sie als Verwalter Zugriff haben, der Spender sieht diese nicht.
*   Sie können Spendenaktionen anlegen und damit kann der Spender zweckgebunden spenden.
*   Spendenaktionen können Sie nach vielen Aspekten auswerten, um beispielsweise aus den A-Spendern Einladungs-Verteilerlisten für die nächsten Spendenaktionen erstellen.
*   Nutzen Sie gesammelte Kontakte und bisherige Spender für die Ansprache bei neuen Spendenaktionen.
 
Newsletter: Nutzen Sie SpendeOnline als Kommunikationsplattform. Senden Sie personalisierte E-Mails oder personalisierte Serienbriefe an Ihre Top-Spender.

*   Versenden Sie über die Plattform spontane Info-Mails an Ihre Spender, oder nur an bestimmte Gruppen (z.B. an alle Handball-Teilnehmer).
*   Versenden Sie über die Plattform ergänzend regelmä??ige E-Mail-Newsletter an Ihre Spender, Interessenten oder nur an bestimmte Gruppen. Ihre Verantwortlichen können im Vorfeld die Beiträge (Ankündigung Veranstaltungen, ...) zusteuern, der Vorstand kann sich auf die redaktionelle Arbeit beschränken. Auf Ihrer Internet-Homepage ist eine Anmeldung/Abmeldung möglich (inkl. Double-Opt-In).

Kasse: Verwalten Sie Ihre Spenden mit SpendeOnline. SEPA-Bankeinzüge und Zahlungsverfolgung ganz einfach.

*   Jede Spende erstellt automatisch einen Buchungsdatensatz.
*   Verwenden Sie als Bezahlart "per Rechnung", "per Bankeinzug" oder einen Partner - auch eine Bezahlung per sofortueberweisung.de und/oder Paypal ist möglich.
*   ??ber HBCI können Sie Ihre Kontoauszüge einlesen, dabei werden offene Rechnungen erkannt und als bezahlt markiert.
*   ??ber SEPA-Dateien sowie online per HBCI können Sie Bankeinzugsdaten an Ihre Bank geben.
*   Erstellen Sie mit einem Klick finanzamtskonforme Spendenquittungen
*   Stellen Sie alle Einnahme/Ausgaben-Belege ein. Sie erhalten dann automatisch eine Einnahme/??berschu??-Rechnung, eine Bilanz, eine Budget-??bersicht und ein Kassenbuch.


== Installation ==

Erste Installation:

*   Installieren und aktivieren Sie das Plugin in WordPress.
*   Fügen Sie das Tag [spendeonline] in Ihre WordPress-Seite ein
*   Testen Sie das System mit der Demo-Webseite (beispielhafte Spenden werden nicht eingezogen!)

Anschlie??end:

*    Melden Sie einen kostenlosen und unverbindlichen Testzugang bei "SpendeOnline.org" für 30 Tage an, Sie gehen keine Verpflichtung ein! Sie erhalten eine Umgebung, zum Beispiel: https://www.spendeonline.org/IHRESEITE/
*   ??ndern Sie den WordPress-Tag in: [spendeonline:IHRESEITE]
